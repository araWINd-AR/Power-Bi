# Power BI Practice Repository

This repository contains my hands-on Power BI practice work and learning files. I created it to organize the concepts I have been learning step by step and to document my progress in data visualization, dashboard development, reporting, and DAX.

The goal of this repository is not only to store files, but to show a consistent learning journey in Power BI through practical exercises and examples.

## Featured Files

Here are some of the key Power BI practice files in this repository:

- **Line Chart.pbix** – basic trend visualization using line charts  
- **Area chart.pbix** – area chart creation and formatting  
- **Drill Through.pbix** – drill-through functionality for interactive reporting  
- **Conditional Formating in Matrix.pbix** – matrix creation and conditional formatting  
- **Calculated Functions in DAX.pbix** – practice with calculated functions in DAX  
- **Implict Measure & Explicit Measure.pbix** – understanding the difference between implicit and explicit measures  

These files highlight different Power BI concepts such as visualization, formatting, interactivity, and DAX-based reporting.

## What This Repository Includes

This repository covers a wide range of Power BI topics, including:

- Charts such as line chart, area chart, pie chart, donut chart, ribbon chart, funnel chart, scatter plot, tree map, waterfall chart, gauge chart, and stacked column chart
- Maps, filled maps, pie maps, and map formatting
- Tables and matrix visuals with formatting, subtotals, totals, number formatting, and conditional formatting
- Slicers such as text slicers, number slicers, and date slicers
- Filters on visual level, page level, and report level
- Drill-through functionality
- Cards, multi-row cards, and text cards
- DAX functions related to text, date, number, and logic
- Calculated columns, measures, implicit measures, and explicit measures
- Sample Excel and CSV datasets used for practice
- Screenshots and output files for better understanding of results

## Purpose of This Repository

The purpose of this repository is to:

- Practice Power BI concepts in a structured and practical way
- Build a stronger foundation in business intelligence and reporting
- Improve data visualization and dashboard design skills
- Keep all learning files in one place for easy reference
- Show consistency in learning and applying Power BI concepts

## My Learning Approach

I built this repository by practicing one concept at a time. Instead of jumping directly into a large dashboard, I focused on understanding each feature individually, such as charts, maps, slicers, filters, matrix formatting, and DAX functions.

This helped me build a clearer understanding of how Power BI works from the basics to more applied reporting tasks. Because of that, many files in this repository focus on a single concept or a small group of related concepts.

## Repository Structure

The repository contains different types of files, including:

- `.png` files for screenshots, outputs, and concept references
